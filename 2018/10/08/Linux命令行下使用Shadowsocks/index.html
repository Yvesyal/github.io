Ubuntu客户端模式

1、登陆

使用拥有管理员权限的账号登陆系统。

2、添加源

由于默认软件源中没有 Shadowsocks-Qt5 的相关信息，因此先运行以下命令，添加作者自己的 PPA 源及密钥。

sudo add-apt-repository ppa:hzwhuang/ss-qt5

3、更新软件源

sudo apt-get update

4、安装 ShadowSocks-qt5 客户端

sudo apt-get install shadowsocks-qt5

稍等片刻即可完成安装。
